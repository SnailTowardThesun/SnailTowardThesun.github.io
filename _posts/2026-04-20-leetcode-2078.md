Two Furthest Houses With Different Colors：寻找两个颜色不同的最远房屋，通过检查首尾元素和中间元素的方法快速求解。

有趣的是，这个问题在现实生活中也有类似的应用场景，比如在城市规划中寻找两个不同功能区之间的最远距离，或者在超市布局中确定不同商品区域的最佳间隔。此外，颜色心理学研究表明，不同颜色的组合会影响人们的空间感知，这也使得这个问题在设计领域具有实际意义。

题目：

https://leetcode.com/problems/two-furthest-houses-with-different-colors/

题目翻译：

在一个数组中，每个元素代表一个房屋的颜色，找出两个颜色不同的房屋之间的最大距离。

示例：

输入：colors = [1,1,1,6,1,1]
输出：3

解题思路：

1. 首先检查首尾元素是否不同，如果不同则直接返回最大距离（数组长度-1）
2. 如果首尾元素相同，遍历中间元素，寻找与首尾不同的元素
3. 计算每个不同元素到首尾的最大距离，取最大值作为结果
4. 时间复杂度：O(n)，空间复杂度：O(1)

代码：

```cpp
class Solution {
public:
    int maxDistance(vector<int>& colors) {
        if(colors[0] != colors[colors.size() - 1]) {
            return colors.size() - 1;
        }
        int ret = INT_MIN;
        for (int i = 1; i < colors.size()-1; i++) {
            if (colors[i] != colors[0]) {
                int tmp = max(i, int(colors.size() - 1 - i));
                ret = max(ret, tmp);
            }
        }

        return ret;
    }
};
```
