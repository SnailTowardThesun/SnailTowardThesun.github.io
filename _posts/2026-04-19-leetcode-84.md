柱状图中最大的矩形：单调栈应用，栈中存索引而非数值。

题目：

给定 n 个非负整数，用来表示柱状图中各个柱子的高度。
每个柱子彼此相邻，且宽度为 1 。
求在该柱状图中，能够勾勒出来的矩形的最大面积。

示例：

示例 1：
输入：heights = [2, 1, 5, 6, 2, 3]
输出：10
解释：最大的矩形面积为 10（高度为 2，宽度为 5）
示例 2：
输入：heights = [2, 4]
输出：4

题目翻译：

给定 n 个非负整数，用来表示柱状图中各个柱子的高度。
每个柱子彼此相邻，且宽度为 1 。
求在该柱状图中，能够勾勒出来的矩形的最大面积。

解题思路：

1. 单调栈法：
- 使用单调递增栈来维护柱子的高度
- 当遇到比栈顶元素低的柱子时，计算栈顶柱子能形成的最大矩形
2. 算法步骤：
- 遍历每个柱子的高度
- 如果当前柱子比栈顶低，则弹出栈顶并计算以其为高的矩形面积
- 最后处理栈中剩余的柱子
3. 复杂度分析：
- 时间复杂度: O(n)，每个柱子最多入栈和出栈一次
- 空间复杂度: O(n)，最坏情况下所有柱子都在栈中

代码：

```cpp
class Solution {
    public:
        int largestRectangleArea(vector<int>& heights) {
            stack<int> st;
            int maxArea = 0;
            int n = heights.size();

            for (int i = 0; i <= n; i++) {
                int curHeight = (i == n) ? 0 : heights[i];
                while (!st.empty() && heights[st.top()] > curHeight) {
                    int h = heights[st.top()];
                    st.pop();
                    int width = st.empty() ? i : i - st.top() - 1;
                    maxArea = max(maxArea, h * width);
                }
                st.push(i);
            }
            return maxArea;
        }
};
```
