> 完全二叉树性质特殊，用位运算优化节点数统计，避免遍历所有节点。

缺失的第一个正数：原地哈希，把数字放到正确的位置上。

题目：

给你一个未排序的整数数组 nums，请你找出其中没有出现的最小的正整数。
你必须实现时间复杂度为 O(n) 并且只使用常数级别额外空间的算法。

示例：

示例 1：
输入：nums = [1, 2, 0]
输出：3
示例 2：
输入：nums = [3, 4, -1, 1]
输出：2
示例 3：
输入：nums = [7, 8, 9, 11, 12]
输出：1

题目翻译：

给你一个未排序的整数数组 nums，请你找出其中没有出现的最小的正整数。
你必须实现时间复杂度为 O(n) 并且只使用常数级别额外空间的算法。

解题思路：

1. 原地哈希法：
- 将数组中每个数放到它对应的位置上（即 nums[i] 放到 nums[nums[i]-1]）
- 遍历数组，第一个不满足 nums[i] == i + 1 的位置就是缺失的最小正数
2. 算法步骤：
- 遍历数组，对于每个数 nums[i]，如果它在 [1, n] 范围内，就将它放到正确的位置
- 再次遍历数组，找到第一个不满足 nums[i] == i + 1 的位置
- 如果都满足，返回 n + 1
3. 复杂度分析：
- 时间复杂度: O(n)，每个数字最多被放置两次
- 空间复杂度: O(1)，原地修改数组

代码：

```cpp
class Solution {
    public:
        int firstMissingPositive(int A[], int n) {
            for (int i = 0, tmp = -1; i < n; i++) {
                while (A[i] - 1 >= 0 && A[i] - 1 < n && A[i] - 1 != i) {
                    swap(A, i, A[i] - 1);
                    if (A[i] == tmp) break;
                    else tmp = A[i];
                }
            }
            for (int i = 0; i < n; i++) {
                if (A[i] - 1 != i) return i + 1;
            }
            return n + 1;
        }

        void swap(int A[], int indx1, int indx2) {
            A[indx1] ^= A[indx2];
            A[indx2] ^= A[indx1];
            A[indx1] ^= A[indx2];
        }
};
```
