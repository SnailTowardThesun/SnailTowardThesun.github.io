四数之和：排序后用双指针，外层两层循环固定两个数，剪枝优化效率高。

题目：

给定一个整数数组 nums 和一个目标值 target，
找出 nums 中的四个整数，使得这四个数的和与 target 相等。
返回所有不重复的四元组。
注意：答案中不可以包含重复的四元组。

示例：

示例 1：
输入：nums = [1, 0, -1, 0, -2, 2], target = 0
输出：[[-2, -1, 1, 2], [-2, 0, 0, 2], [-1, 0, 0, 1]]
示例 2：
输入：nums = [2, 2, 2, 2], target = 8
输出：[[2, 2, 2, 2]]

题目翻译：

给定一个整数数组 nums 和一个目标值 target，
找出 nums 中的四个整数，使得这四个数的和与 target 相等。
返回所有不重复的四元组。
注意：答案中不可以包含重复的四元组。

解题思路：

1. 排序 + 双指针法（扩展三数之和）：
- 对数组进行排序
- 固定两个数，然后使用双指针寻找另外两个数
2. 算法步骤：
- 对数组进行排序
- 遍历数组，固定 nums[i] 和 nums[j]
- 使用双指针 left = j + 1，right = n - 1
- 如果四数之和等于 target，添加到结果集
- 如果四数之和大于 target，right--
- 如果四数之和小于 target，left++
3. 复杂度分析：
- 时间复杂度: O(n^3)，其中 n 是数组的长度
- 空间复杂度: O(1)，只使用常数级别的额外空间（不考虑输出空间）

代码：

```cpp
class FourSumSolution {
    public:
        vector<vector<int>> fourSum(vector<int>& nums, int target) {
            set<vector<int> > res;
            sort(nums.begin(), nums.end());
            for (int i = 0; i < int(nums.size() - 3); ++i) {
                for (int j = i + 1; j < int(nums.size() - 2); ++j) {
                    int left = j + 1, right = nums.size() - 1;
                    while (left < right) {
                        int sum = nums[i] + nums[j] + nums[left] + nums[right];
                        if (sum == target) {
                            vector<int> out;
                            out.push_back(nums[i]);
                            out.push_back(nums[j]);
                            out.push_back(nums[left]);
                            out.push_back(nums[right]);
                            res.insert(out);
                            ++left; --right;
                        }
                        else if (sum < target) ++left;
                        else --right;
                    }
                }
            }
            return vector<vector<int> >(res.begin(), res.end());
        }
    };

    FourSumSolution s;
    
    // 测试用例 1
    vector<int> nums1 = {1, 0, -1, 0, -2, 2};
    auto result1 = s.fourSum(nums1, 0);
    EXPECT_EQ(result1.size(), 3);
    
    // 测试用例 2
    vector<int> nums2 = {2, 2, 2, 2};
    auto result2 = s.fourSum(nums2, 8);
    EXPECT_EQ(result2.size(), 1);
    EXPECT_EQ(result2[0], vector<int>({2, 2, 2, 2}));
    
    // 测试用例 3
    vector<int> nums3 = {0};
    auto result3 = s.fourSum(nums3, 0);
    EXPECT_EQ(result3.size(), 0);
};
```
