删除排序数组中的重复项：双指针技巧，快指针探索，慢指针守门。

题目：

给你一个有序数组 nums，请你原地删除重复出现的元素，使每个元素只出现一次，返回移除后数组的新长度。
不要使用额外的数组空间，必须在原地修改输入数组并在使用 O(1) 额外空间的条件下完成。

示例：

示例 1：
输入：nums = [1,1,2]
输出：2
解释：函数应返回新长度 2，并且 nums 中的前两个元素必须是 1 和 2。不需要考虑数组中超出新长度后面的元素
示例 2：
输入：nums = [0,0,1,1,1,2,2,3,3,4]
输出：5
解释：函数应返回新长度 5，并且 nums 中的前五个元素必须是 0, 1, 2, 3, 4。不需要考虑数组中超出新长度后面的元素

题目翻译：

给你一个有序数组 nums，请你原地删除重复出现的元素，使每个元素只出现一次，返回移除后数组的新长度。
不要使用额外的数组空间，必须在原地修改输入数组并在使用 O(1) 额外空间的条件下完成。

解题思路：

1. 双指针法：
- 使用两个指针，一个用于遍历数组，一个用于记录不重复元素的位置
2. 算法步骤：
- 如果数组长度小于等于 1，直接返回长度
- 初始化 i = 0，j = 1
- 当 j < n 时，比较 nums[i] 和 nums[j]
- 如果相等，j++；如果不等，将 nums[j] 复制到 nums[i+1]，i++，j++
- 返回 i + 1
3. 复杂度分析：
- 时间复杂度: O(n)
- 空间复杂度: O(1)

代码：

```cpp
class Solution {
    public:
        int removeDeplicate(int A[], int n) {
            if (n <= 1) return n;
            int i = 0, j = 1;
            while (j < n) {
                if (A[i] == A[j]) j++;
                else A[++i] = A[j++];
            }
            return i + 1;
        }
};
```
