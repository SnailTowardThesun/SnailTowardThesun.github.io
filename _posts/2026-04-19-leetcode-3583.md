使字符串成为回文的最少插入次数：动态规划求最小插入。

题目：

给你一个整数数组 nums。如果三元组 (i, j, k) 满足以下条件，则称其为特殊三元组：
- 0 <= i < j < k < nums.length
- nums[i] + nums[k] == 2 * nums[j]
返回特殊三元组的数量。由于答案可能很大，返回结果对 10^9 + 7 取模。

示例：

示例 1：
输入: nums = [6,3,6]
输出: 1
解释: 只有三元组 (0, 1, 2) 满足条件：6 + 6 = 2 * 3
示例 2：
输入: nums = [0,1,0,0]
输出: 2
解释: 满足条件的三元组有 (0,1,2) 和 (0,1,3)

题目翻译：

给你一个整数数组 nums。如果三元组 (i, j, k) 满足以下条件，则称其为特殊三元组：
- 0 <= i < j < k < nums.length
- nums[i] + nums[k] == 2 * nums[j]
返回特殊三元组的数量。由于答案可能很大，返回结果对 10^9 + 7 取模。

解题思路：

1. 问题转化：
- 条件 nums[i] + nums[k] == 2 * nums[j] 可以改写为 nums[k] = 2 * nums[j] - nums[i]
- 对于固定的中间位置 j，需要找到满足条件的 (i, k) 对
2. 算法步骤：
- 使用哈希表存储每个数值出现的所有位置（预处理）
- 遍历每个可能的中间位置 j（从 1 到 n-2）
- 对于每个 j，遍历所有 i < j：
a) 计算 target = 2 * nums[j] - nums[i]
b) 在哈希表中查找 target 的所有位置
c) 统计其中位置大于 j 的数量
- 累加所有满足条件的三元组数量
3. 优化策略：
- 使用二分查找优化位置查询
- 哈希表中存储的位置列表是有序的（按遍历顺序添加）
- lower_position 函数找到第一个大于 j 的位置，计算满足条件的 k 的数量
4. 复杂度分析：
- 时间复杂度: O(n^2 log n)，对于每个 (i,j) 对使用二分查找
- 空间复杂度: O(n)，哈希表存储所有位置
5. 边界处理：
- 注意处理 target = 0 的特殊情况（位置计算略有不同）
- 结果对 10^9 + 7 取模

代码：

```cpp
class Solution {
       public:
        static std::size_t lower_position(const std::vector<int>& vec, int key) noexcept {
            std::size_t low = 0;
            std::size_t high = vec.size();  // 开区间 [low, high)

            while (low < high) {
                std::size_t mid = low + (high - low) / 2;
                if (vec[mid] <= key)
                    low = mid + 1;  // 舍弃左半段
                else
                    high = mid;  // 舍弃右半段（含 mid）
            }
            return low;  // low 即为所求位置
        }

        int specialTriplets(std::vector<int>& nums) {
            if (nums.size() < 2) {
                return 0;
            }

            int64_t result = 0;

            std::unordered_map<int, std::vector<int> > container;

            for (auto i = 0; i < nums.size(); ++i) {
                if (container.find(nums[i]) == container.end()) {
                    container[nums[i]] = std::vector<int>();
                }

                container[nums[i]].push_back(i);
            }

            for (auto& it : container) {
                // sort it->second
                std::sort(it.second.begin(), it.second.end());
            }

            for (auto i = 1; i < nums.size() - 1; i++) {
                auto target = nums[i] * 2;
                auto it = container.find(target);
                if (it == container.end()) {
                    continue;
                }

                auto pos = lower_position(it->second, i);
                if (target == 0) {
                    result += int(pos) * (it->second.size() - pos - 1);
                } else {
                    result += int(pos) * (it->second.size() - pos);
                }
            }

            return int(result % 1000000007);
        }
};
```
