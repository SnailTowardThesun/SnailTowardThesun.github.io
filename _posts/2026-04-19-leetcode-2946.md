> 最长公共子序列变体常用于序列比对，DNA比对是典型应用场景。

矩阵中的循环移位：模拟旋转操作，检测四个方向的循环。

题目：

给你一个下标从 0 开始的 m x n 整数矩阵 mat 和一个整数 k。
如果满足以下条件，称矩阵是循环移位的：
- 矩阵中的每一行都可以循环右移 k 次后得到原矩阵的对应行
如果矩阵 mat 是循环移位的，返回 true，否则返回 false。
行的循环右移操作：
- 将行中每个元素向右移动一位，最后一个元素移到第一个位置
- 执行 k 次这样的操作

示例：

示例 1：
输入: mat = [[1,2,3],[4,5,6],[7,8,9]], k = 4
输出: false
示例 2：
输入: mat = [[5,4,5,10,5]], k = 9
输出: false
解释: 循环右移 9 次等价于右移 9 % 5 = 4 次，无法得到原行

题目翻译：

给你一个下标从 0 开始的 m x n 整数矩阵 mat 和一个整数 k。
如果满足以下条件，称矩阵是循环移位的：
- 矩阵中的每一行都可以循环右移 k 次后得到原矩阵的对应行
如果矩阵 mat 是循环移位的，返回 true，否则返回 false。
行的循环右移操作：
- 将行中每个元素向右移动一位，最后一个元素移到第一个位置
- 执行 k 次这样的操作

解题思路：

1. 关键观察：
- 循环右移 k 次等价于右移 k % n 次（n 是列数）
- 如果 k % n == 0，则每行都是原行，直接返回 true
- 否则需要检查每行是否满足循环移位条件
2. 算法步骤：
- 计算实际移动步数：actual_step = k % n
- 对于每一行的每个元素 mat[i][j]
- 检查是否等于 mat[i][(j + actual_step) % n]
- 如果所有元素都满足，返回 true；否则返回 false
3. 数学原理：
- 循环右移 k 次后，原位置 j 的元素会移动到位置 (j + k) % n
- 因此，移位后的行与原行相等，当且仅当 mat[i][j] == mat[i][(j+k)%n]
4. 复杂度分析：
- 时间复杂度: O(m * n)，需要遍历整个矩阵
- 空间复杂度: O(1)，只使用常数额外空间

代码：

```cpp
class Solution {
    public:
        bool areSimilar(vector<vector<int>>& mat, int k) {
            int  n = mat.size();
            int  m = mat[0].size();

            auto actual_step = k % m;
            for (int i = 0; i < n; i++) {
                for (int j = 0; j < m; j++) {
                    if (mat[i][j] != mat[i][(j + actual_step)%m]) {
                        return false;
                    }
                }
            }
            return true;
        }
};
```
