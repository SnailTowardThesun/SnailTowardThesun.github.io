最大子数组和：动态规划找最大和 Kadane算法真巧妙。

题目：

给你一个整数数组 nums，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

示例：

示例 1：
输入：nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]
输出：6
解释：连续子数组 [4, -1, 2, 1] 的和最大，为 6
示例 2：
输入：nums = [1]
输出：1
示例 3：
输入：nums = [0]
输出：0

题目翻译：

给你一个整数数组 nums，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

解题思路：

1. 动态规划（Kadane 算法）：
- 定义 dp[i] 为以第 i 个元素结尾的最大子数组和
- dp[i] = max(nums[i], dp[i-1] + nums[i])
- 维护一个全局最大和
2. 算法步骤：
- 初始化 global = nums[0]，local = nums[0]
- 遍历数组，更新 local 和 global
- 返回 global
3. 复杂度分析：
- 时间复杂度: O(n)
- 空间复杂度: O(1)

代码：

```cpp
class MaxSub {
    public:
        int maxSubArray(vector<int>& nums) {
            int global = nums[0];
            int local = nums[0];
            int size = nums.size();
            for (int i = 1; i < size; i++) {
                local = nums[i] > local + nums[i] ? nums[i] : local + nums[i];
                global = local > global ? local : global;
            }
            return global;
        }
    };

    MaxSub s;
    
    // 测试用例 1
    vector<int> nums1 = {-2, 1, -3, 4, -1, 2, 1, -5, 4};
    EXPECT_EQ(s.maxSubArray(nums1), 6);
    
    // 测试用例 2
    vector<int> nums2 = {1};
    EXPECT_EQ(s.maxSubArray(nums2), 1);
    
    // 测试用例 3
    vector<int> nums3 = {0};
    EXPECT_EQ(s.maxSubArray(nums3), 0);
    
    // 测试用例 4
    vector<int> nums4 = {-1};
    EXPECT_EQ(s.maxSubArray(nums4), -1);
    
    // 测试用例 5
    vector<int> nums5 = {-2, -1};
    EXPECT_EQ(s.maxSubArray(nums5), -1);
};
```
