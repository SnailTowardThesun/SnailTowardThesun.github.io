寻找两个正序数组的中位数：二分查找的经典应用，折半查找效率高。

题目：

给定两个正序数组 nums1 和 nums2，长度分别为 m 和 n，
请找出这两个正序数组的中位数。
整体运行时间复杂度应为 O(log(m+n))。

示例：

示例 1：
输入：nums1 = [1, 3], nums2 = [2]
输出：2.00000
解释：合并数组 = [1, 2, 3]，中位数是 2
示例 2：
输入：nums1 = [1, 2], nums2 = [3, 4]
输出：2.50000
解释：合并数组 = [1, 2, 3, 4]，中位数是 (2 + 3) / 2 = 2.5

题目翻译：

给定两个正序数组 nums1 和 nums2，长度分别为 m 和 n，
请找出这两个正序数组的中位数。
整体运行时间复杂度应为 O(log(m+n))。

解题思路：

1. 合并数组法（简单但不是最优）：
- 合并两个有序数组
- 直接计算中位数
2. 算法步骤：
- 使用双指针遍历两个数组
- 比较两个数组的元素，较小的被添加到结果中
- 处理较长数组的剩余部分
- 根据合并后数组的长度计算中位数
3. 复杂度分析：
- 时间复杂度: O(m+n)，遍历两个数组
- 空间复杂度: O(1)，只使用常数级别的额外空间

代码：

```cpp
class Solution {
    public:
        double findMediaSortedArrays(int A[], int m, int B[], int n) {
            if (!m && !n) return NULL;
            int tmp = (int)(m+n)/2;
            int result1 = 0;
            int result2 = 0;
            int a = 0, b = 0;

            for (int i = 0; i < tmp; i++) {
                if (a < m && b < n)
                    A[a] < B[b] ? result1 = A[a++] : result1 = B[b++];
                else
                    a < m ? result1 = A[a++] : result1 = B[b++];
            }
            if ((m+n) % 2) {
                if (a < m && b < n)
                    A[a] < B[b] ? result1 = A[a++] : result1 = B[b++];
                else
                    a < m ? result1 = A[a++] : result1 = B[b++];
                return result1;
            } else {
                if (a < m && b < n)
                    result2 = A[a] < B[b] ? A[a++] : B[b++];
                else
                    result2 = a < m ? A[a++] : B[b++];

                return (result2 + result1) / 2.0;
            }
        }
};
```
