判断矩阵经轮转后是否一致：枚举四种旋转角度，逐一比较。

题目：

给你两个大小为 n x n 的二进制矩阵 mat 和 target。
现以 90 度顺时针轮转矩阵 mat，并检查轮转后矩阵是否与 target 相等。
如果可以，返回 true；否则，返回 false。
矩阵的 90 度顺时针轮转：
- 矩阵元素 (i, j) 会移动到位置 (j, n-1-i)

示例：

示例 1：
输入: mat = [[0,1],[1,0]], target = [[1,0],[0,1]]
输出: true
解释: 顺时针轮转 90 度一次可以使 mat 与 target 相等。
示例 2：
输入: mat = [[0,1],[1,1]], target = [[1,0],[0,1]]
输出: false
解释: 无法通过轮转使 mat 与 target 相等。
示例 3：
输入: mat = [[0,0,0],[0,1,0],[0,0,0]], target = [[0,0,0],[0,0,0],[0,0,0]]
输出: false
解释: 无法通过轮转使 mat 与 target 相等。

题目翻译：

给你两个大小为 n x n 的二进制矩阵 mat 和 target。
现以 90 度顺时针轮转矩阵 mat，并检查轮转后矩阵是否与 target 相等。
如果可以，返回 true；否则，返回 false。
矩阵的 90 度顺时针轮转：
- 矩阵元素 (i, j) 会移动到位置 (j, n-1-i)

解题思路：

1. 关键观察：
- 矩阵可以轮转 0°、90°、180°、270° 四种状态
- 只需检查这四种状态是否与 target 相等
2. 优化策略：
- 优化1: 使用数学公式直接计算旋转后的位置，避免多次 push_back
顺时针90度旋转公式: (i, j) -> (j, n-1-i)
- 优化2: 直接比较，无需真正旋转矩阵
根据旋转次数计算原坐标，直接与 target 比较
这样可以节省创建新矩阵的空间
3. 算法步骤：
- 检查 mat 和 target 大小是否相同
- 尝试 0°、90°、180°、270° 四种旋转
- 对每种旋转，使用坐标映射直接比较
- 如果任意一种匹配，返回 true
4. 复杂度分析：
- 时间复杂度: O(4 * n^2) = O(n^2)，只需常数次遍历矩阵
- 空间复杂度: O(1)，无需额外矩阵空间

代码：

```cpp
class Solution {
       public:
        // 优化1: 使用数学公式直接计算旋转后的位置，避免多次push_back
        vector<vector<int>> rotate(const vector<vector<int>>& mat) {
            int n = mat.size();
            if (n == 0) return {};

            vector<vector<int>> ret(n, vector<int>(n));
            for (int i = 0; i < n; i++) {
                for (int j = 0; j < n; j++) {
                    // 顺时针90度: (i, j) -> (j, n-1-i)
                    ret[j][n - 1 - i] = mat[i][j];
                }
            }
            return ret;
        }

        // 优化2: 直接比较，无需真正旋转矩阵
        bool check(const vector<vector<int>>& mat, const vector<vector<int>>& target, int rotation) {
            int n = mat.size();
            for (int i = 0; i < n; i++) {
                for (int j = 0; j < n; j++) {
                    int ni = i, nj = j;
                    // 根据旋转次数计算原坐标
                    for (int r = 0; r < rotation; r++) {
                        int ti = ni;
                        ni = nj;
                        nj = n - 1 - ti;
                    }
                    if (mat[ni][nj] != target[i][j]) {
                        return false;
                    }
                }
            }
            return true;
        }

        bool findRotation(vector<vector<int>>& mat, vector<vector<int>>& target) {
            if (mat.size() != target.size()) return false;

            // 尝试0°, 90°, 180°, 270°四种旋转
            for (int k = 0; k < 4; k++) {
                if (check(mat, target, k)) {
                    return true;
                }
            }
            return false;
        }
};
```
